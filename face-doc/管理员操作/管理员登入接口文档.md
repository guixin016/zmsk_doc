### 管理员登入接口文档 ###
---

### 登入操作


|method|URL|
|---|---|
|POST|http://face.zmsk.com/face/admin/login?account=6&password=|

参数说明

|参数名|描述|
|---|---|
|account|账号|
|password|密码|

返回值说明

成功
```json
{
  "code": 200,
  "data": {
    "account": "admin",
    "adminId": 1,
    "adminName": "admin"
  },
  "msg": "success"
}
```

失败

```json
{
  "code": 401,
  "msg": "Invalid account or password"
}
```

返回值参数说明

|参数名|描述|
|---|---|
|account|账号|
|adminId|管理员Id|
|adminName|管理员名|

### 导出认证记录

|URL|face.doingo.com/api/manager/authertication/record/export?organizationId=&dateStr=|
|---|---|
|method|GET|

参数说明

|参数名|描述|
|---|---|
|organizationId|组织Id|
|dateStr|日期字符串 如2018-07-08|

失败

```json
{
	"code": 50001,
	"exception": "ExprotExcelException",
	"msg": "未查询到导出数据",
	"ressource": ""
}
```

成功 直接下载文件

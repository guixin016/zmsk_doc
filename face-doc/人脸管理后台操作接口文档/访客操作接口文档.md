### 访客操作接口文档 ###
----

### 登记成访客

|URL|face.doinggo.com/api/manager/visitor/register?id=&remark=|
|---|---|
|method|POST|

参数描述

|参数名|描述|
|---|---|
|id|认证记录主键Id|
|remark|备注信息|

返回值

```json
{
  "code": 200,
  "msg": "success"
}
```

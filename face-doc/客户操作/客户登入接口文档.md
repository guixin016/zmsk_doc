### 登入操作接口文档

|method|URL|
|---|---|
|POST|http://face.zmsk.com/face/customer/login?account=6&password=|

参数说明

|参数名|描述|
|---|---|
|account|账号|
|password|密码|

返回值说明

成功
```json
{
  "code": 200,
  "data": {
    "account": "18105035196",
    "userId": 1,
    "userName": "warrior"
  },
  "msg": "success"
}
```

失败

```json
{
  "code": 401,
  "msg": "Invalid account or password"
}
```

返回值参数说明

|参数名|描述|
|---|---|
|account|账号|
|userId|用户Id|
|userName|用户名|

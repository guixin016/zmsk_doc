### 客户查询授权码信息

|method|URL|
|---|---|
|GET|http://face.zmsk.com/customer/action/code/query?customerId=&actionCode=|

参数说明

|参数名|描述|
|---|---|
|customerId|客户Id|
|actionCode|授权码|

返回值

```json
{
  "code": 200,
  "data": [
    {
      "actionTime": 1519723053,
      "code": "673dseqt",
````

参数说明

|参数名|描述|
|---|---|
|customerId|客户Id|

返回值

```json
{
  "code": 200,
  "data": 8,
  "msg": "success"
}
```

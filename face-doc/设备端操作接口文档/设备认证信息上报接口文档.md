### 设备认证信息上报接口文档 ###
---

###  添加认证记录

|URL|face.doinggo.com/api/device/authentication/record/add?name=&idNumber=&nation=&address=&avatar=&sex=&type=&result=&deviceNumber=&groupId=|
|---|---|
|mehtod|POST|

参数说明

|参数名|描述|
|---|---|
|name|名称|
|idNumber|身份证号|
|nation|民族|
|address|身份证地址|
|avatar|头像|
|sex|性别1：女2：男|
|type|类型1：人证，2人脸|
|result|认证结果 0：未通过1：通过|
|deviceNumber|身边登入账号|
|groupId|所属分组Id,不是必填，默认为0|

返回值

成功

```json
{
  "code": 200,
  "msg": "success"
}
```

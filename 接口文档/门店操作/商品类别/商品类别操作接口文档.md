### 商品操作接口文档 ###
----

### 1、添加商品类别

>添加商品类别

|URL|http://www.zmsk.com/api/commodity/category/create?storeId=&categoryName=&categorySort=|
|---|---|
|method|POST|

参数说明

|参数名|描述|
|---|---|
|storeId|门店Id|
|categoryName|类别名称|
|categorySort|类别排序值,非必填，默认值为0|

返回值

成功
```
{
  "code": 200,
  "msg": "success"
}
```

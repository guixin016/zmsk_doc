### 会员角色权限操作接口文档 ###
---

### 绑定会员角色

|URL|http://www.zmsk.com/api/store/member/role/bind?memberId=&roleId=|
|---|---|
|method|POST|


|参数名|参数描述|
|---|--|
|memberId|会员Id|
|roleId|角色Id|


返回值

```json
{
  "code": 200,
  "msg": "success"
}
```

### 解绑会员角色

|URL|http://www.zmsk.com/api/store/member/role/unbind?memberId=&roleId=|
|---|---|
|method|POST|


|参数名|参数描述|
|---|--|
|memberId|会员Id|
|roleId|角色Id|


返回值

```json
{
  "code": 200,
  "msg": "success"
}
```

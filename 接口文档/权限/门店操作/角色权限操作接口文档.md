### 角色权限操作接口文档 ###
---

### 绑定角色权限

|URL|http://www.zmsk.com/api/store/role/permission/bind?roleId=&permissionId=|
|---|---|
|method|POST|

|参数名|参数描述|
|---|--|
|roleId|角色Id|
|permissionId|权限Id|

返回值

```json
{
  "code": 200,
  "msg": "success"
}
```


### 解绑角色权限

|URL|http://www.zmsk.com/api/store/role/permission/unbind?roleId=&permissionId=|
|---|---|
|method|POST|

|参数名|参数描述|
|---|--|
|roleId|角色Id|
|permissionId|权限Id|

返回值

```json
{
  "code": 200,
  "msg": "success"
}
```

### 根据角色Id获取权限信息

|URL|http://www.zmsk.com/api/store/role/permission/query?roleIds=|
|---|---|
|method|GET|


|参数名|参数描述|
|---|--|
|roleIds|角色Ids,多个id使用逗号分隔|

返回值

```json
{
  "code": 200,
  "data": [
    {
      "name": "test1",
      "percode": "test:*",
      "permissionId": 1,
      "roleId": 2,
      "url": "/ddss"
    },
    {
      "name": "test1",
      "percode": "test:*",
      "permissionId": 1,
      "roleId": 3,
      "url": "/ddss"
    },
    {
      "name": "test模块",
      "percode": "2222",
      "permissionId": 2,
      "roleId": 3,
      "url": "22222"
    }
  ],
  "msg": "success"
}
```

### 点击呼叫操作接口

> 点击呼叫操作接口

|URL|http://www.zmsk.com/api/rest/pad/notify/add?storeId=&tableId=|
|---|---|
|method|POST|

参数说明

|参数名|参数描述|
|---|--|
|storeId|门店Id|
|tableId|桌号|

返回值

成功

```
{
  "code": 200,
  "msg": "success"
}
```

可能的错误

|httpCode|说明|
|---|---|
|400|参数错误|
|405|Method Not Allowed，HTTP请求方法错误|
|415|Unsupported Media Type，没有设置请求头的Content-type|
|500|内部服务器错误|
